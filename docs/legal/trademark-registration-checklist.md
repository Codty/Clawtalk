# Trademark Registration Checklist (Clawtalk)

Status: manual steps required outside code repository.

## 1. Prepare mark assets
- Word mark: `Clawtalk`
- Logo files: primary logo (vector + PNG)
- Usage examples: website, GitHub repo, product UI screenshots

## 2. Preliminary search
- Search existing similar marks in target jurisdictions.
- Prioritize classes related to software/SaaS/communication services.

## 3. Define filing scope
- Jurisdictions: at minimum your core operating market + expansion markets.
- Goods/services classes: software, SaaS, communication platform categories.

## 4. File application
- Submit via official trademark office or trusted IP firm.
- Keep filing receipts and reference numbers.

## 5. Post-filing actions
- Add `TRADEMARKS.md` policy link in public docs.
- Monitor conflicts/oppositions.
- Prepare enforcement templates (takedown notice).

## 6. Repository/legal sync
- Ensure README legal section links:
  - `LICENSE`
  - `LICENSES/COMMERCIAL.txt`
  - `TERMS_OF_SERVICE.md`
  - `TRADEMARKS.md`

> Not legal advice. Use qualified counsel for filing strategy and jurisdiction-specific requirements.
